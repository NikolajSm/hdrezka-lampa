# HDRezka Plugin for Лампа TV

Плагін для перегляду контенту з HDRezka.ua на Android TV через додаток "Лампа".

## Особливості
- Перегляд фільмів, серіалів, мультфільмів та аніме
- Підтримка української локалізації
- Автоматичний вибір якості відео
- Пошук контенту
- Кешування даних
- Підтримка проксі для обходу обмежень

## Встановлення

### Спосіб 1: Через файл ZIP
1. Завантажте останню версію плагіна
2. У додатку "Лампа" перейдіть у Налаштування → Плагіни
3. Виберіть "Встановити з файлу"
4. Виберіть завантажений ZIP файл

### Спосіб 2: Через GitHub
1. Додати репозиторій плагіна в Лампа
2. Знайти "HDRezka UA" у списку доступних плагінів
3. Встановити

## Налаштування

### Основні налаштування:
- **Проксі**: для обходу геоблокування
- **Якість відео**: вибір бажаної якості
- **Кешування**: для швидшого доступу

### Розширені налаштування:
- User-Agent для запитів
- Таймаут з'єднання
- Кількість спроб підключення
- Логування для налагодження

## Використання

1. Запустіть додаток "Лампа"
2. Перейдіть у розділ "Плагіни"
3. Запустіть "HDRezka UA"
4. Оберіть категорію або скористайтесь пошуком
5. Насолоджуйтесь переглядом

## Проблеми та вирішення

### Плагін не завантажує контент:
- Перевірте підключення до інтернету
- Спробуйте увімкнути проксі
- Перевірте наявність оновлень

### Відео не відтворюється:
- Перевірте доступність HDRezka.ua
- Спробуйте змінити якість відео
- Перезавантажте плагін
